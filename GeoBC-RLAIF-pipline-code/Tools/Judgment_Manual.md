# “类内变化 vs 类间变化”标注判定手册

## 一、标注目标

本手册用于双时相遥感图像中变化区域的判定，核心任务不是“看到哪里不一样”，而是判断：

1. 前后时相是否属于**相同二级类土地利用类别**；
2. 若视觉上不同但对应的二级类土地利用类别未发生目标跃迁，则判为**类内变化 / 非目标变化**；
3. 若差异仅由阴影、光照、物候、积水、配准偏移等引起，则判为**伪变化**。

------

## 二、判定结果必须使用的四个标签

每个候选变化区域最终必须落入以下五类之一：

### 1. `NO_CHANGE`

前后属于**同一类别**，且没有证据表明土地利用功能发生改变。
 这类样本即使颜色、纹理、含水量、作物长势、边界清晰度发生变化，也不作为目标变化。

### 2. `INTRA_CHANGE`

前后属属于**不同类别**。
 例如同属耕地，但由“水田”变为“水浇地”或“旱地”；同属交通运输用地，但由“乡村道路”变为“其他交通运输用地”。这类应视为类间变化。耕地、交通运输用地等二级边界在摘录中有明确文字定义。 或例如耕地变建设用地、林地变草地、水域变建设用地。该类始终属于**目标变化**。

### 3. `PSEUDO_CHANGE`

视觉差异存在，但无法支持土地利用类别发生变化。
 典型来源：阴影变化、太阳高度角变化、物候变化、积水/退水、收割、临时裸露、传感器差异、配准误差等。

### 4. `UNCERTAIN`

存在类别边界模糊、分辨率不足、遮挡严重、证据冲突等情况，无法稳定判断。
 必须给出不确定原因，并降低置信度，必要时建议实地核查。

------

## 三、总原则：先“独立分类”，再“比较变化”

### 原则 1：先判 t1 类别，再判 t2 类别，最后才判变化

禁止从“看起来变了”直接跳到“发生了类间变化”。
 正确流程是：

- 先对 t1 单独判别类别；
- 再对 t2 单独判别类别；
- 最后依据“前类 vs 后类”给出变化标签。

### 原则 2：以**土地利用类型**为主，不以单一视觉表象为主

同一地块即使颜色、湿度、纹理、表面覆盖状态变化明显，只要其土地利用功能未发生目标跃迁，就不应直接标为类间变化。

### 原则 3：中间状态优先服从标准定义

例如处于施工中的地块，不应只因呈裸地或杂乱纹理，就草率标成“裸土地”或“其他建设用地”；若其在监测期内处于施工、采矿、拆迁等人为扰动状态，应优先考虑“人为扰动地”。摘录已明确把矿山、采石场、在建工程、拆迁场地等纳入“人为扰动地”。

------

## 四、正式判定流程

## 第 1 步：锁定候选变化单元

以分割掩码或专家圈定区域为单位，不以整图为单位直接判别。
 每个变化单元必须绑定：

- mask 或 polygon掩模或多边形
- t1、t2 的局部裁剪
- 周边上下文
- 绝对位置与相对位置

## 第 2 步：对 t1 做独立分类

标注员必须回答：

- 一级类是什么？
- 二级类是什么？
- 证据是什么？

证据应尽量来自：

- 形状与边界
- 连通关系
- 周边依附对象
- 场景功能
- 稳定结构特征

## 第 3 步：对 t2 做独立分类

要求与 t1 相同，不得被 t1 结果“牵着走”。

## 第 4 步：比较前后类别

按以下规则输出：

- 二级相同 → `NO_CHANGE`
- 一级相同、二级不同 → `INTRA_L1_CHANGE`
- 一级不同 → `INTER_L1_CHANGE`

## 第 5 步：执行伪变化排除

必须逐项检查：

- 是否只是阴影位置变化？
- 是否只是作物生长期不同？
- 是否只是季节性积水？
- 是否只是边界轻微漂移或配准误差？
- 是否只是同类内部整治、翻耕、收割、压实？

若排除后无足够证据支撑类别跃迁，则改判 `PSEUDO_CHANGE` 或 `UNCERTAIN`。

## 第 6 步：给出置信度

推荐 1–100 分。
 80 分以上：类别边界较清晰；
 60–79 分：存在一定不确定性；
 60 分以下：高混淆风险，建议复核或实地核查。

------

## 五、关键类别的具体判定规则

## 5.1 耕地内部：水田 / 水浇地 / 旱地

摘录中定义：
 水田是种植水生作物的耕地，且**包括水旱轮作地**；水浇地是有稳定水源和灌溉设施、主要种植旱作物的耕地；旱地则主要依靠天然降水、无灌溉设施。

### 判定规则

- 若只是同一块水田在不同季节表现为蓄水、插秧、收割后裸露，仍应优先判为**同一二级类内部状态变化**，尤其文件已明确“水旱轮作地也属于水田”。
- 若证据表明地块长期从水生作物种植转为旱作灌溉耕作，可判 `INTRA_L1_CHANGE`（耕地一级内跨二级）。
- 若耕地转为建设用地、交通运输用地、水域等，则判 `INTER_L1_CHANGE`。

### 易错提醒

- “有水”不等于“水域”；
- “暂时不见作物”不等于“不是耕地”；
- “翻耕后变裸”不等于“裸土地”。

------

## 5.2 园地内部：果园 / 茶园 / 其他园地

摘录中把果园、茶园和其他种植桑、橡胶、咖啡、药材等多年生作物的园地分开定义。

### 判定规则

- 同一园地内因生长期、修剪、疏植造成外观差异，不作为目标变化。
- 果园改茶园、茶园改其他园地，记为 `INTRA_L1_CHANGE`。
- 园地改建设用地、林地、草地等，记为 `INTER_L1_CHANGE`。

### 易错提醒

- 零散树木不能直接判果园；
- 周边农业组织结构和成片种植格局比单棵树冠更重要。

------

## 5.3 林地内部：有林地 / 灌木林地 / 其他林地

摘录中规定：有林地为郁闭度 ≥ 0.2 的乔木林，灌木林地要求灌木覆盖度 ≥ 40%，其他林地包括疏林地、未成林地、迹地、苗圃等。

### 判定规则

- 森林因采伐、更新、幼林培育导致的状态变化，若符合“迹地、未成林地、苗圃”等情形，应优先在林地一级内部判别，不要直接判为非林地。摘录已明确这些属于“其他林地”。
- 有林地 ↔ 灌木林地 ↔ 其他林地，为 `INTRA_L1_CHANGE`。
- 林地改草地、建设用地、耕地等，为 `INTER_L1_CHANGE`。

### 易错提醒

- 单次时相中树冠稀疏，不一定就是草地；
- 采伐后裸露，不必然等于裸土地。

------

## 5.4 草地内部：天然牧草地 / 人工牧草地 / 其他草地

摘录中天然牧草地强调自然草本植被、用于放牧或割草；人工牧草地强调人工种植牧草；其他草地要求树木郁闭度 < 0.1、表层为土质、以草本为主且不用于畜牧业。

### 判定规则

- 草地枯荣、季节变色、放牧强度变化，不自动构成类别跃迁。
- 天然牧草地 ↔ 人工牧草地 ↔ 其他草地，为 `INTRA_L1_CHANGE`。
- 草地改建设用地、耕地、林地等，为 `INTER_L1_CHANGE`。

### 易错提醒

- 仅凭“规则纹理”不能武断判人工牧草地；
- 与林地的边界需结合郁闭度与优势植被结构判断。摘录给出了林地与其他草地的关键阈值线索。 

------

## 5.5 建设用地内部：城镇建设用地 / 农村建设用地 / 人为扰动地 / 其他建设用地

摘录明确：
 城镇建设用地位于城镇建成区内；农村建设用地位于农村居民点内部；人为扰动地包括采矿、采石、在建工程、拆迁场地等监测期内活动地；其他建设用地包括独立于城镇村庄之外的工业、堆场、盐田、军事、科研、名胜古迹等。

### 判定规则

- 施工中的地块优先考虑“人为扰动地”，不能因其视觉上像裸地，就先判“裸土地”。
- 村庄内部宅基地、养殖设施、乡村公共设施，应优先判“农村建设用地”。
- 独立于城乡居民点之外的工业生产、物资堆放等，判“其他建设用地”。

### 变化标签

- 城镇建设 ↔ 农村建设 ↔ 人为扰动地 ↔ 其他建设用地，为 `INTRA_L1_CHANGE`。
- 建设用地与耕地/林地/水域/交通运输用地之间转换，为 `INTER_L1_CHANGE`。

### 易错提醒

- 临时工地 ≠ 永久建设完成态；
- 拆迁中的地块 ≠ 裸土地；
- 村边独立厂区不必然属于农村建设用地。

------

## 5.6 交通运输用地内部：乡村道路 / 其他交通运输用地

摘录对这组边界给得非常关键：
 乡村道路是农村地区服务农业生产的**未铺装道路**；而除乡村道路以外的建成交通用地均属“其他交通运输用地”，且**农村地区的铺装道路也归入后者**。

### 判定规则

- 农村地区的道路，先看是否铺装。
- 未铺装、服务农业生产的道路，判“乡村道路”；
- 已铺装道路，即便位于农村，也应判“其他交通运输用地”。

### 变化标签

- 未铺装乡村道路 → 铺装道路：`INTRA_L1_CHANGE`
- 耕地 → 乡村道路 / 其他交通运输用地：`INTER_L1_CHANGE`

### 易错提醒

- “在农村”不是“乡村道路”的充分条件；
- 是否铺装是这里最重要的硬边界之一。

------

## 5.7 水域及水利设施用地：河流湖泊水库坑塘 / 沼泽地

摘录把河流、湖泊、水库、坑塘、滩涂和水工建筑统一纳入一类，同时把频繁积水、长期过湿并生长湿生植物的地块定义为沼泽地。

### 判定规则

- 临时积水农田、季节性水膜、短期洪涝，不应轻易判为“河流湖泊水库坑塘”或“沼泽地”。
- 若地块呈长期水淹/过湿，并有湿生植被支撑，可考虑“沼泽地”。
- 稳定坑塘、水面扩张、河道整治等可在该一级类内部细分。

### 变化标签

- 河湖库塘 ↔ 沼泽地：`INTRA_L1_CHANGE`
- 水域改建设/耕地/交通等：`INTER_L1_CHANGE`

### 易错提醒

- “有水”与“长期水域功能”不是一回事；
- 沼泽地判定要有“频繁水淹/过湿 + 湿生植物”双重依据。

------

## 5.8 其他土地：盐碱地 / 沙地 / 裸土地 / 裸岩石砾地

摘录中：沙地要求地表为沙漠、植被覆盖 < 5%；裸土地为土质地表且植被覆盖 < 5%；裸岩石砾地要求裸岩石砾覆盖 > 70%；盐碱地强调表层盐碱聚积及耐盐植被。

### 判定规则

- “植被少”不自动等于沙地，先看基底是沙还是土。
- “裸露”不自动等于裸岩石砾地，需有高比例岩石砾石覆盖。
- 若地块为施工扰动导致的短期裸露，应优先与“人为扰动地”区分。建设用地摘录已对人为扰动状态给出定义。

### 变化标签

- 盐碱地 / 沙地 / 裸土地 / 裸岩石砾地内部转换：`INTRA_L1_CHANGE`
- 其他土地与耕地、建设、水域等转换：`INTER_L1_CHANGE`

------

## 六、最重要的“非目标变化”规则

以下情况默认优先考虑 `NO_CHANGE` 或 `PSEUDO_CHANGE`，除非有强证据证明类别跃迁：

### 1. 物候变化

作物从生长旺盛到收割后裸露，林草地从繁茂到枯黄。

### 2. 阴影变化

高大建筑、山体、太阳高度变化造成亮度差异。

### 3. 季节性积水

耕地、低洼地短时积水，不自动转为水域或沼泽。

### 4. 同类内部工程整治

道路压实、地块整形、田块平整、林地采伐后更新。

### 5. 配准误差

边缘整体平移、双边错位、细长目标偏移。

------

## 七、分层判定口径：必须同时给出一级与二级结论

建议输出两个字段：

- `label_L1`
- `label_L2`

示例：

- 水田 → 水浇地
  - `label_L1 = 类内变化`
  - `label_L2 = 类间变化`
- 乡村道路（未铺装） → 其他交通运输用地（铺装）
  - `label_L1 = 类内变化`
  - `label_L2 = 类间变化`
- 耕地 → 建设用地
  - `label_L1 = 类间变化`
  - `label_L2 = 类间变化`

这一步能彻底解决“到底算不算变化”的口径混乱。
